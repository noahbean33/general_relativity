Lecture 6: Fields — International Winter School on Gravity and Light 20151. Introduction: From Points to FieldsThe concept of a field was introduced into physics by Michael Faraday. While Faraday lacked formal mathematical training by modern standards, he developed the notion of fields to describe physical phenomena intuitively through pictures. Modern differential geometry provides the rigorous mathematical framework—the theory of bundles—to underpin Faraday's visual intuition.+2Previously, our technical focus was restricted to a single tangent space and vectors at a single point. As physicists, however, we are interested in Vector Fields, where a vector is associated with every point on a manifold.+12. The Theory of BundlesTo mathematically define a field, we use the structure of a bundle.Definition: A BundleA bundle is a triple $(E, \pi, M)$ consisting of:Total Space ($E$): A smooth manifold.Base Space ($M$): A smooth manifold.Projection Map ($\pi$): A surjective, smooth map $\pi: E \to M$.The FiberFor any point $p \in M$, the fiber over $p$ is the pre-image of the set $\{p\}$ under the projection map $\pi$:
$$F_p = \pi^{-1}(p)$$In our construction, the fiber represents the collection of all possible values a field can take at that specific point $p$.3. Sections: The Mathematical Definition of a FieldA section $\sigma$ of a bundle is the formal definition of a field.+1Definition: SectionA section $\sigma$ is a map $\sigma: M \to E$ that satisfies a compatibility condition: the projection of the section must return the original base point.
$$\pi \circ \sigma = \text{id}_M$$Intuition: A section "picks" exactly one element from the fiber over every point in the manifold.+1Types of Fields: The type of field (vector, co-vector, tensor, etc.) is determined by the nature of the fiber.+1Quantum Mechanics: In QM, a wave function is not strictly a function but a section of a complex line bundle over the base space.4. Detailed Construction of the Tangent Bundle ($TM$)To understand Vector Fields, we must construct the Tangent Bundle of a smooth manifold $(M, \mathcal{O}, \mathcal{A})$.+1A. As a SetWe define $TM$ as the disjoint union of all tangent spaces $T_pM$ for all $p \in M$.
$$TM = \bigsqcup_{p \in M} T_pM$$Using a "disjoint" union ensures that even if vectors from different tangent spaces look identical, they remain unique to their specific base point.+1B. The Projection MapThe map $\pi: TM \to M$ takes an element $X \in TM$ (a "nose") and maps it to the unique point $p \in M$ to which that vector belongs.C. Topology and Smooth StructureTopology: We equip $TM$ with the initial (coarsest) topology such that the projection map $\pi$ is continuous.Smooth Atlas: We construct an atlas $\mathcal{A}_{TM}$ on the bundle using the atlas $\mathcal{A}$ of the base manifold $M$.+2Dimension: If the base manifold $M$ has dimension $d$, the total space $TM$ has dimension $2d$.D. Chart Map ConstructionFor a chart $(U, x)$ on $M$, we define a chart map $\xi$ on $TU \subset TM$ that captures both the base point coordinates and the vector components:
+1$$\xi(X) = (x^1(\pi(X)), \dots, x^d(\pi(X)), dx^1(X), \dots, dx^d(X))$$The first $d$ components are the coordinates of the base point.The second $d$ components are the vector components with respect to the coordinate basis $\frac{\partial}{\partial x^i}$.+15. Smooth Vector Fields and Algebraic ModulesA smooth vector field $\xi$ is defined as a smooth section of the tangent bundle $TM$.The $C^\infty(M)$-moduleThe set of all smooth vector fields, denoted $\Gamma(TM)$, possesses a specific algebraic structure.+1Ring of Functions: Smooth functions $C^\infty(M)$ form a ring, not a field, because not every non-zero function has a multiplicative inverse (functions can be zero at specific points).Module: Because we multiply vector fields by these functions (scaling the field differently at every point), $\Gamma(TM)$ is a $C^\infty(M)$-module, not a vector space.Note: Every vector space has a basis, but not every module has a basis. This means we cannot always find a set of global vector fields to expand every other field.+1The Hairy Ball TheoremThe sphere $S^2$ is a classic counter-example. You cannot "comb" a hairy sphere—any smooth vector field on $S^2$ must vanish at some point. Because a basis vector cannot be zero, $S^2$ cannot have a global basis of vector fields.+16. Tensor FieldsA tensor field is a generalization of the vector field concept.Definition: $(r, s)$-Tensor FieldAn $(r, s)$-tensor field $T$ is a $C^\infty(M)$-multilinear map that eats $r$ co-vector fields and $s$ vector fields to produce a smooth function:
$$T: \underbrace{\Gamma(T^*M) \times \dots}_{r} \times \underbrace{\Gamma(TM) \times \dots}_{s} \to C^\infty(M)$$Crucially, this map must be linear with respect to scaling by functions, not just real numbers.Example: The Gradient FieldFor a smooth function $f$, the gradient $df$ is a $(0, 1)$-tensor field (a co-vector field). Its action on a vector field $\xi$ produces a function $df(\xi) = \xi f$.
